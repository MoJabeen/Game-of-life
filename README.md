## Rules of the game

1. LIVE cell with 0 or 1 neighbour = DEAD
2. LIVE cell with 2 or 3 neighbours = LIVE
3. LIVE cell with > 3 neighbours = DEAD 
4. DEAD cell with 3 neighbours = LIVE
